Fire Detection and Suppression System

Problem Statement

The aim of this project is to build a semi automatic fire detection and suppression system using an Arduino Uno microcontroller board, enabled with Bluetooth to send appropriate notifications remotely via a customised Android Application.

Solution and Approach

ATo achieve this, an Arduino microcontroller is interfaced with a number of temperature sensors (LM35) and humidity sensors (DHT-11) in various areas to monitor an enclosed area of 1m X 1m X 1m approximately to provide a zonal abstraction. The collected data is processed to check for fire conditions. If fire conditions arise, a buzzer is used to alert the surroundings, an appropriate message is displayed on an adjoining OLED display, and a notification is sent to the phones of the people in charge via a bluetooth enabled customised Android app. Relay switches are provided which can be used to trigger efficient extinguishers as soon as a fire is detected.

Skills

Here are the skills I used and learned during this project:

  • Sensors and Actuators Interfacing

    • Impact

    This project, with the purpose of installation in high fire risk situations such as army tanks, etc, provides an alternate solution to generic smoke detector systems, to alleviate risk in situations where burning materials don't necessarily produce smoke.
