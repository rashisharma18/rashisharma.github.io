Penetration Testing: Windows System

About the Project

In this "Capture The Flag" project, a Penetration test was conducted on a system of Windows Machines, and a singular Ubuntu Server.
The objective of the Test was to exploit the system to discover the hidden flag and map out the security vulnerabilites encountered on the system. The final results were presented in a thorough Technical Report, outlined with an Executive Summary, including specific security recommendations with a detailed plan of action to mitigate the vulnerabilities present on the system.

Solution and Approach

The system was provided as a series of Virtual Machine images, and consisted of an assortment of Windows Operating Systems and Servers. The system also consisted of an Ubuntu Server hosting a static HTML Web Page and a Cloud Storage Bucket. A Kali machine was used as the attacking machine.

Approach

Network Enumeration: First, a complete network and port analysis was conducted to map IP addresses, and disocver the services running on each machine.
Webpage Reconnasaince: The static web page hosted on the Ubuntu Server was analyzed to disocver web page vulnerabilties.
Initial Access: Inital access was gained into the system by exploiting an existing Eternal Blue vulnerability using Metasploit.
Credentialed Access and Lateral Movement: Brute Forcing techniques were used to crack NLTM Hashes and gain Credentailed Access into the system and the cloud to discover the Flag.

Skills

Here are the skills I used and learned during this project:

  • Tools Used: NMAP, Metasploit, John, Hashcat, SMBClient, VMWare.
  • Cloud Resources Used: Amazon Web Services, S3 Buckets.
  • Operating Systems: Windows, Linux, Kali.

Impact

This project helped me get hands-on experience on performing penetration tests on Windows Systems. I discovered the Security Posture of various Windows programs such as Active Directories, Windows Backups and Third Party Password Management Tools such as KeePass, and provide specific security recommendations to mitigate the risks.
