Results Tabulation and Notification

Problem Statement

My team at Hewlett Packard Enterprise ran 100+ Regression and Robustness Tests everyday on more than 3 systems. The results for these tests had to be manually updated to an Excel sheet which was later sent out to stakeholders. This project was meant to automation the population, tabulation and notification of the test results, so they could be delivered upon completion to the necessary stakeholders without manual intervention.

Solution and Approach

A Python script was created to parse results from JSON files for each test system and populate to a Microsoft Excel Sheet and create Pivot Tables and corresponding charts based on the Day's result. The next script would take the charts populated in the latest excel sheet and send out an email with the tabulated results to the stakeholders. This integration was done on Jenkins.
