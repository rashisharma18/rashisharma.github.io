Forensic Analysis: Windows System

About the Project

In this project, an end-to-end Digital Forensic Analysis was conducted on a Windows Hard Drive Image. The goal of this analysis was to recover malware stored on the drive, analyze the suspected binaries, and draw up a comprehensive report highlighting the functionality of the malware and security recommendations to prevent further exploitation.

Solution and Approach

Disk Image Analysis: Hard Disk Image was analyzed using the software Autospy.
Malware Discovery and Traffic Analysis: After combing through multiple stored files and folders, a few obfuscated binaries were discovered and analyzed and traffic analysis was conducted using Wireshark.
Unveiling encrypted code:The evidence obtained via traffic analysis pointed to encrypted data on the drive which was decrypted using VeraCrypt, which contained deobfuscated versions of the malware.

Impact

This project helped me get hands-on experience on performing Forensic Analysis on an Image, look for evidence in packet captures and learn the basics of reverse engineering obfuscated code to gain a basic understanding of the functionality of a binary. I also discovered new tools: Autopsy and VeraCrypt.
