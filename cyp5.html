Cloud Migration using AWS

About the Project

The goal of this project was to migrate an on-premises Web Application architecture to the Cloud, to overcome the existing security vulnerabilities existing in the system such as Denial of Service, and provide a Proof of Concept for the proposed design.

Solution and Approach

To migrate the application, a thorough analysis of Amazon Web Services Products was done, with an effort to combat security vulnerabilities such as Denial of Service, lack of Backup, Access Management, etc. The proposed design was implemented on AWS to prove the accuracy of the model.

Skills

Here are the skills I used and learned during this project:

  • AWS: EC2, S3, VPC, Backup, Aurora

    • Impact

    This project helped me get hands-on experience working with Cloud Migration and AWS. I was able to study the various AWS products, their security implications and design a secure web application hosted on the cloud.
