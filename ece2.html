VerTech: Smart Farming Solution

Problem Statement

To design a fully automated smart farming solution equipped with a sensor-actuator network to monitor growth factors such as humidity, temperature, and UV radiation. This project was a submission to the IICDC 2018, a nation-wide innovation challenge sponsored by Texas Instruments and Indian Institute of Management, Banglore. The project qualified for quarterfinals amongst 500+ entries.

Solution and Approach

A low-power sensor node solution was designed, that allowed users to actively control the status of their vertical farming setup, while providing a configurable automated water dispersion, temperature, humidity and UV radiation control system tailored to the optimal growth of the plants. The user would interact with the system via an Android app and analyze the efficiency of the same using a cloud-based dashboard.
